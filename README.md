*A batch data redaction tool.*

The goal of this software is to allow any user to quickly and easily batch-redact information from a large number of files.

## Supported File Extensions
| File Type | Supported |
|-----------|:---------:|
| .txt      |     ❌     |
| .png      |     ❌     |
| .jpg      |     ❌     |
| .jpeg     |     ❌     |
| .pdf      |     ❌     |
| .docx     |     ❌     |
| .gif      |     ❌     |
